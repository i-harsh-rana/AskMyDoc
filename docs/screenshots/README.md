# Screenshots

These images are referenced by the root [`README.md`](../../README.md).

The committed PNGs are **placeholders** so the README renders without broken
image links. To populate them with real screenshots, run the app
(see *Getting Started* in the root README) and replace each file, keeping the
same filename:

| File | Page | Suggested capture |
|------|------|-------------------|
| `landing.png` | `/` | Hero / landing page |
| `upload.png` | `/chat` | Empty state with the PDF dropzone |
| `chat.png` | `/chat` | A conversation — questions + Markdown answers with the provider badge |
| `profile.png` | `/profile` | User profile and the LLM provider cards/modal |

Recommended size: ~1200×750px (16:10) for consistent rendering in the tables.
